# AWS cases

Para poder seguir estos notebooks es necesario un dominio activo de SageMaker, un rol de IAM con permisos adecuados y una cuenta de AWS con cuotas de servicios suficientes (en cuentas nuevas personales suele ser habitual que no se disponga de cuotas de Processing Jobs).

Estás diseñados para trabajarse dentro de AWS Academy, tras una configuración de Sagemaker adecuada.

La ejecución de estos notebooks incurrirá en gastos fuera de la capa gratuita y potencialmente gastos recurrentes.
